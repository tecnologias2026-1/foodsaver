POLÍTICA DE PRIVACIDAD – FOODSAVER

En FoodSaver respetamos la privacidad de los usuarios. Esta aplicación no requiere registro ni inicio de sesión para su uso.

1. Datos que recopilamos

FoodSaver no solicita datos personales identificables como nombre, correo electrónico o contraseña.

Sin embargo, podemos recopilar información no personal, como:

  • Datos de uso dentro de la aplicación
  • Información técnica del dispositivo (tipo de dispositivo, sistema operativo)
  • Preferencias de uso (si el usuario guarda configuraciones dentro de la app)

2. Finalidad del uso de datos

La información recopilada se utiliza para:

  • Mejorar el funcionamiento de la aplicación
  • Analizar el uso de las funcionalidades
  • Detectar errores y optimizar el rendimiento

3. Almacenamiento de datos

Los datos pueden almacenarse de forma local en el dispositivo del usuario o mediante servicios externos necesarios para el funcionamiento de la app.

4. Compartición de datos

No compartimos información personal con terceros.

En caso de utilizar servicios externos (por ejemplo, herramientas de análisis), estos pueden procesar datos técnicos de forma anónima.

5. Derechos del usuario

Dado que no se recopilan datos personales identificables, no se gestionan perfiles de usuario.

El usuario puede eliminar los datos almacenados desinstalando la aplicación.

6. Cambios en la política

FoodSaver se reserva el derecho de modificar esta política en cualquier momento.
